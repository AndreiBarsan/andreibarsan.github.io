One of the requirements for my Bachelor's Thesis, which was written in English, was for the table of contents to be available in both English and Romanian.

Since I was using LaTeX (XeLaTeX, to be more precise), I decided there had to be a way to do this in a clever fashion (i.e. without having to manually create the Romanian ToC from scratch).

And, as it turns out, there is a way to do this, using tocloft! The original question I found while googling the issue is located here: http://latex-community.org/forum/viewtopic.php?f=5&t=5438&p=84468#p84468

I've cleaned up the code a little and updated the label names accordingly (to make it easy for anyone to change them into any language).

\newcommand{\numecuprins}{Cuprins}
\newlistof{cuprins}{tce}{\numecuprins}
% definition of the commands used for the Romanian ToC
\newcommand\capitol[1]{%
\addcontentsline{tce}{chapter}{\protect\makebox[1.3em][l]{\thechapter}#1}}
\newcommand\sectiune[1]{%
\addcontentsline{tce}{section}{\protect\makebox[2.8em][l]{\thesection}#1}}
\newcommand\subsectiune[1]{%
\addcontentsline{tce}{subsection}{\protect\makebox[3em][l]{\thesubsection}#1}}

% rest of shitte

clearpage
\listofcuprins
